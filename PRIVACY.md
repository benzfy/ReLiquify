# 隐私政策 (Privacy Policy)

**生效日期：** 2025年12月17日

ReLiquify ("我们") 非常重视用户的隐私。本隐私政策旨在说明我们在您使用 ReLiquify Chrome 扩展程序（"本扩展"）时如何收集、使用和保护您的信息。

## 1. 数据收集与使用

### 1.1 本地处理
ReLiquify 的核心功能是将网页内容转换为 Markdown 格式。这一过程完全在您的浏览器本地进行（Client-side）。我们不会在后台收集、存储或传输您的浏览历史、页面内容或任何个人身份信息。

### 1.2 "发送到云端" 功能（可选）
本扩展提供一个可选的 "发送到云端 (Send to Cloud)" 功能。
- **数据传输**：仅当您**主动点击**该菜单项时，本扩展才会将当前页面提取的 Markdown 内容发送到您配置的服务器地址（默认为 `https://limhub.xiaoluxue.com`）。
- **认证信息**：为了使用此功能，您需要在设置中配置 API Key。该 Key 仅用于验证您在目标服务器上的身份。

### 1.3 数据存储
我们使用 Chrome 的 `storage.sync` API 来存储您的偏好设置（例如：API Key 和 API URL）。这些数据会通过您的 Google 账户在不同设备间同步，但不会被我们或第三方读取。

## 2. 权限说明

为了实现功能，本扩展需要以下权限：
- **activeTab**: 允许扩展访问您当前正在查看的标签页，以便提取内容。
- **scripting**: 允许扩展向页面注入必要的转换脚本 (Readability.js, turndown.js)。
- **downloads**: 允许扩展将生成的 Markdown 文件下载到您的设备。
- **contextMenus**: 用于在右键菜单中添加 "提取并复制" 等选项。
- **storage**: 用于保存您的配置信息。
- **所有网站访问权限 (<all_urls>)**: 允许扩展在任何网页上运行内容提取脚本。

## 3. 第三方服务
如果您使用了 "发送到云端" 功能，数据将依照您配置的目标服务器的隐私政策进行处理。对于默认配置的 LimHub 服务，请参考其相应的服务条款。

## 4. 变更
我们可能会不时更新本隐私政策。任何更改都将在此页面上发布。
